# bth650-group1
ar glasses ui implementation

## installation

clone the repo
```
$ git clone https://github.com/BenLee8602/bth650-group1.git <<<PROJECT_FOLDER>>>
$ cd <<<PROJECT_FOLDER>>>
```
  
install dependencies
```
$ npm i
```
  
start the app
```
$ npm start
```
  
open a browser and go to `http://localhost:3000/` to use the app
